SNAF Module v1.2

Product Listing for Specials, New, All and Featured Products

This add-on is a re-working of the Single Listing Template is targeted at Zen 1.5.0. It will not work with previous versions of Zen without modification to the sql statements.

What This Module Does

It adds 'column grid' functionality that allows products to be displayed in columns rather than rows in the product_listing display. It adds the following configuration options to configuration>>Product Listing

  • Product Listing - Columns Per Row
  • Product Listing - Layout Style
  • Use Product Listing Filter for Column Display
  • Use Style Changer for Product Listing

The Product Listing Filter is a dropdown menu that allows the user to sort the listing on certain criteria. It is similar to the dropdowns used in the default all products listing.

The Style Changer is a button on the Product Listing page that allows the user to change styles.

It adds the following configuration options to configuration>>All Listing

  • Use Product Listing Format for All Products
  • Use Product Listing Filter for All Products in SNAF

These allow the store owner to choose to use the product_listing display format or the default display format and to choose whether to display the dropdown listing filter.

It adds the following configuration options to configuration>>New Listing

  • Use Product Listing Format for New Products
  • Use Product Listing Filter for New Products in SNAF

These allow the store owner to choose to use the product_listing display format or the default display format and to choose whether to display the dropdown listing filter.

It adds the following configuration options to configuration>>Featured Listing

  • Use Product Listing Format for New Products
  • Use Product Listing Filter for New Products in SNAF

These allow the store owner to choose to use the product_listing display format or the default display format and to choose whether to display the dropdown listing filter.

It creates a new menu item as configuration>>Specials Listing It adds the following configuration options to configuration>>Specials Listing

  • Include Salemaker Items in Specials Listing
  • Use Product Listing Format for Specials Listing
  • Use Product Listing Filter for Specials in SNAF

These allow the store owner to choose to use the product_listing display format or the default display format and to choose whether to display the dropdown listing filter.

The store owner can also choose to include salemaker items in the 'specials' categorisation.

It may be noted that there is also a category sort facility built into the code that allows the specials to be filtered on the basis of categories using the get parameter sale_category ( a url might look like this index.php?main_page=specials&sale_category=1). Which can be useful at times.

Installation

This module will only work with zen Cart 1.5.0 at the time of writing. (Jan 2012)

For previous versions you would need to adapt the sql statements that are run on installation.

  • BACKUP YOUR SITE AND DATABASE AND PUT THE BACKUP SOMEWHERE SAFE
  • If you have decided to skip the step above, please change your mind
  • Download the package and un-zip it to your computer.
  • In the 'Installation Files' folder make the following changes
    -- rename the YOUR_ADMIN folder to match your installation
    -- rename the includes/templates/YOUR_TEMPLATE folder to match your active template
    -- rename the includes/modules/YOUR_TEMPLATE folder to match your active template
  • If you are using a language other than English you may want to create a version of the file YOUR_ADMIN/includes/languages/english/extra_definitions/snaf.php for your own language.
  • Upload the files in the Installation Files folder to your server. They are structured so that the upload should be simple.
  • The first time that you access an admin page the installation of the database changes will be actioned. You will receive a message at the top of the page saying that it has happened correctly.

Version Notes

A style changer feature has been introduced. You have the choice of using this on the product listing pages or not. If you use it an icon will appear on those pages that allows the user to change between row style and column style. Yes, I am not a graphic designer. So, the icon is not beautiful but you will probably wish to change it yourself to match your own template. To do so just replace the two image files that are in the buttons directory. If you wish to change from a jpg then you will need to edit the language file additionally.

An inner container has been placed in the column view 'boxes'. This is primarily to allow easier styling to achieve certain effects. For instance, a gutter or spacing between the 'boxes' and a border around the 'boxes' on the product listing can be achieved with the stylesheet rule:

.centerBoxContentsProducts .innerColumn{
margin:5px;
border:1px solid #aaa;
}

Uninstall

There are two steps to uninstalling this module.

  1. First unistall the database changes. This can be done manually, or by re-uploading an altered version of the file YOUR_ADMIN/includes/auto_loaders/config.snaf.php. Simply open the file in your text editor and uncomment the line that is clearly shown. Then upload this file back to your server in the autoloaders directory. Access a admin page and the database changes will be uninstalled.
  2. Remove all the files from the installation replacing them, where appropriate, with the corresponding files from your backup.
